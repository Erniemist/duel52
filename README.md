# Installation
- Clone the repo
- Install [python 3.12](https://www.python.org/downloads/release/python-3128/)
- `python -m venv .venv`
- POSIX
  - bash: `source .venv/bin/activate`
  - fish: `source .venv/bin/activate.fish`
- Windows: `.venv\Scripts\activate.bat`
- `pip install -r requirements.txt`
- `python client.py`
